# Daksh.AI 🎯

**Daksh.AI** is an intelligent career development platform designed for students, 
freshers, and early-career professionals. It leverages AI to help users build 
job-ready profiles, learn the right skills, and stand out in competitive job markets.

## ✨ Features

- 📄 **AI Resume Maker (Pro)** — Build ATS-compatible, professionally formatted 
  resumes with education, skills, experience, and certification sections using 
  curated Google Fonts typography.

- 📊 **AI Resume Summarizer** — Upload your existing resume and get an instant 
  AI-generated summary of your profile, with the ability to manage uploaded files.

- 🗺️ **Personalized Learning Path** — Get a tailored skill roadmap based on your 
  dream job. Each course links directly to high-view YouTube tutorials.

- 💼 **Proof-of-Work Portfolio** — Add GitHub repos, Figma designs, live websites, 
  and video demos to build a verifiable digital portfolio.

- 🌐 **Bilingual Support** — Full English and Hindi (हिंदी) language toggle.

- 🔐 **Firebase Authentication** — Secure sign-up/login with Google OAuth and 
  email/password support.
